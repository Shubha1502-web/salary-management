\# Salary Management Tool



A full-stack HR salary management tool for managing 10,000+ employees.



\## Live Demo

> Record a short Loom video and paste the link here after recording



\## Tech Stack



\### Backend

\- Node.js + Express + TypeScript

\- Prisma ORM + SQLite

\- Vitest + Supertest for testing



\### Frontend

\- React + TypeScript (Create React App)

\- Plain CSS with custom properties

\- Axios for API calls



\## Features

\- Add, view, edit and delete employees

\- Search by name or email

\- Filter by country and job title

\- Paginated table (20 per page, 10,000 employees)

\- Salary insights dashboard — min, max, average per country

\- Average salary by job title with vs org average comparison

\- Fast seed script using batch inserts (500 rows per batch)
